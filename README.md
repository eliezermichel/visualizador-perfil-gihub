# Visualizador de Perfil do GitHub

Este projeto é um aplicativo web que permite visualizar perfis de usuários do GitHub de forma simples e interativa.

## Funcionalidades
- Busca de usuários do GitHub pelo nome de usuário
- Exibição de informações do perfil: avatar, nome, bio, seguidores, repositórios, etc.
- Interface responsiva e moderna

## Tecnologias Utilizadas
- HTML
- CSS (incluindo animações e responsividade)
- JavaScript
- Consumo da API pública do GitHub

## Estrutura do Projeto

## Como Usar
1. Clone o repositório:
	 ```bash
	 git clone <url-do-repositorio>
	 ```
2. Abra o arquivo `index.html` em seu navegador.
3. Digite o nome de usuário do GitHub que deseja pesquisar.

## Observações
- O projeto utiliza apenas tecnologias front-end.
- Não é necessário instalar dependências.
